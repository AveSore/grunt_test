# grunt_test
目录结构为
>
```
|- build 
  |-- 子目录全部运行自动生成
|- dev
  |-- api
  |-- components
  |-- static
    |--- js
    |--- css
    |--- font
    |--- img
    |--- less
    |--- sass
    |--- lcss运行后自动生成存放由less目录中文件生成css代码的目录，主要是为了与原有的css区分开
    |--- scss运行后自动生成存放由sass目录中文件生成css代码的目录，主要是为了与原有的css区分开
  |- template
    |-- jade
    |-- view
```

##### grunt搭建请参照简书[Grunt安装与环境配置](http://www.jianshu.com/p/a339f2dc3823)
